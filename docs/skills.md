# LumiShop Project Skills

This repository includes internal agent guidance files to help Claude, Codex, Gemini, and similar coding agents work consistently.

## Files
- `CLAUDE.md` — repo-specific guidance for Claude-style coding assistants
- `AGENTS.md` — general agent guidance and repo direction
- `.skills/lumishop-dev/SKILL.md` — reusable internal skill for LumiShop development tasks

## What these files do
They help coding agents understand:
- the product goal
- the architecture direction
- the sequencing of implementation work
- where to put business logic vs framework logic

## Current focus
The current priority is turning the repo into a clean full-stack Shopify app foundation with strong backend boundaries and a future-ready FE/BE split.
