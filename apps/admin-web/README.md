# LumiShop Admin Web

This is the initial admin frontend scaffold for LumiShop.

## Purpose
- provide an embedded-style admin experience for merchants
- surface product sync, video generation jobs, themes, and settings
- evolve into a Shopify Polaris + App Bridge frontend

## Current state
- built with React + Vite
- static dashboard scaffold only
- not yet connected to Shopify auth or backend APIs

## Next steps
- add Shopify App Bridge
- add Polaris UI
- add real routing
- connect to backend generation APIs
- add product and job data fetching
