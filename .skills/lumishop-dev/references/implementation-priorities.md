# Implementation Priorities

## Recommended build order
1. Establish project structure
2. Add Shopify auth and webhook verification
3. Add persistence for shops, products, and jobs
4. Build template rendering pipeline
5. Add admin UI for preview and generation
6. Add storefront embed/publish flow
7. Add analytics and experiments
8. Add premium AI-assisted generation later

## Why this order
This sequence reduces risk, keeps the app demoable early, and prevents expensive AI/rendering work from blocking the basic Shopify flow.
