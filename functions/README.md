# LumiShop Functions API

Firebase Cloud Functions backend for LumiShop.

## Responsibilities
- health endpoint
- video generation requests
- Shopify webhooks
- template resolution
- future auth/session hooks
- future render orchestration

## Current state
- Express on Firebase Functions v2
- mock generation pipeline scaffold
- event and product theme skills included

## Next steps
- Shopify OAuth
- webhook verification
- persistence
- FFmpeg rendering integration
